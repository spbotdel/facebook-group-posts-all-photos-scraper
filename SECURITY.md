# Security Policy

## Supported version

The Apify Store version of this Actor is the supported runtime.

## Reporting a vulnerability

Please report security issues privately to:

`spbotdel@gmail.com`

Do not open public issues containing:

- API tokens;
- cookies or login sessions;
- private group content;
- personal contact data;
- raw payloads that include sensitive data.

## Responsible use boundary

This Actor is built for public Facebook group data. It does not require Facebook credentials and is not intended to bypass login, checkpoints, group privacy settings, rate limits, or access controls.
